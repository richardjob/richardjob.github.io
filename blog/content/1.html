What is Cloud Computing?

Cloud computing is the on-demand delivery of compute power, database storage, application resources and other IT resources through a cloud service platform via the Internet. These services are often offered in a pay as you go manner (i.e) pay per hour of usage. They offer high level of security following a shared responsibilty model were AWS manages the security of the cloud and the coustomer has the control over the security in the cloud.


Why Amazon Web Services?

AWS is global leader in the field of cloud computing and cloud related services. Using AWS will reduce the cost of setting up on-premise setup of servers and storages which may become useless if there is decrease in the requirements. There is no long term contracts or upfront commitments. With AWS you take advantage of scalable, relaible, and secure global infrastructure.

AWS for Students and Educators

With the increasing demand for cloud employees, AWS Educate provides an academic gateway for the next generation of IT and cloud professionals. AWS Educate is Amazon’s global initiative to provide students and educators with the resources needed to accelerate cloud-related learning.

AWS Educate provides Institutions, Educators and Students with resources for cloud related training. There is an option to get started with a Starter Account in AWS. This is a great deal when compared to the usual account which needs a credit card to sign up. The Starter Account is valid upto the graduation day. There are pathways for which digital badges are provided upon completion.

AWS Educate also offers a Job board were students can search and apply for cloud jobs and internships around the world. There are much more exciting things to know, check out AWS educate
